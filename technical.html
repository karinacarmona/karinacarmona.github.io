Website Compare

On Tumblr, I'm not looking for much of anything.ON Modcloth it's really simple with thier drop down menus and quick view feature. The Week is kind of hard to navigate from another page. You need to go back to the homepage to find what you're looking for. It is so easy to browse through Tumblr's content. That's why it's so hard to leave! Modcloth is pretty easy they have easy filters to help me find what I'm looking for. The week is fun to browse through since I'm normally not looking for anything.

CSS Stuff


I've been struggling with a lot of CSS stuff. So here's positioning. What a beast.

Relative


This means that it is relative to itself. So if there are no other positioning attributes applied to the element (top,left,bottom,right) then nothing will happen to the element. What it means that when you say 10px left. It will be 10px left of where it would NORMALLY be. Helps altering things when they aren't lining up like you thought they would.
Also if there is no z-index value this relative element will be positioned on top of all other elements.It also limites the scope of the absolutely positioned child elements.

Absolute


Literally means that you can place the element EXACTLY where you want it. This will be relative to the parent element, with no parent element it will position itself according to the page.

Fixed


This is relative to the viewport or the browser window itself. It's fixed and will stay right where it is when the page is scrolled through.

JavaScript Questions



Why is Javascript popular?


When the internet was first becoming popular, the most popular broswer was Netscape Navigator. And Netscape used JavaScript for dynamic front-end web development. So if you were a web developer you had to use JavaScript to do what you gotta do. Therefore JavaScript became popular and important out of necessity and prevalence during the internet's early days.

What are great things about JavaScript


JavaScript is so open in what you can write. It will take almost anything and try to make it work. Thus it's fairly easy to use. JavaScript is straightforward in it DOM. Also, the source code is a free for all so if you want to see the way a developer did something all you have to do is right-click.

#map! Method

Array#map! is kind of like a loop. It will go through each element of the array and carry out the code block that's attached to the method call. That code block should be written with the intention to output a different number than the original element in the array. #map! should be used then you want to manipulate/translate elements in an array.
